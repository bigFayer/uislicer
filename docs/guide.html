1. 为什么换方案？

旧方案的根本缺陷：让AI"猜"像素坐标

旧方案（auto_slicer.py / ui_slicer_workflow.py）的工作方式是：

游戏UI图片
→
发送给AI大模型
(GPT-4V / GLM)
→
AI返回JSON
{"x":100, "y":200}
→
OpenCV按坐标裁切

问题：语言模型不是空间定位器。它返回的 [x:100, y:200, w:120, h:40] 是近似值，永远无法精确到像素级。金币切偏、图标残缺、坐标漂移都是必然结果。

新方案：让分割模型直接输出像素掩码

游戏UI图片
→
Grounding DINO
文本引导检测
→
SAM 2
像素级分割
→
透明PNG导出

关键区别：不再"猜坐标"，而是用专业检测模型定位 + 分割模型生成每个像素的掩码。圆形金币、不规则装饰都能精确裁切。

2. 工作原理

Phase 1: Grounding DINO（文本引导检测）
  输入: 图片 + 文本提示 "button . icon . coin . panel"
    ↓
  Grounding DINO (IDEA-Research/grounding-dino-tiny, ~600MB)
    ↓
  输出: 每个匹配元素的精确边界框 (x1,y1,x2,y2)
    ↓
  [释放显存] 卸载 Grounding DINO

Phase 2: SAM 2（像素级分割）
  输入: 原图 + Phase 1 的边界框
    ↓
  SAM 2 (sam2.1-hiera-tiny, ~80MB)
    ↓
  输出: 每个元素的像素级二值掩码 (H×W bool数组)
    ↓
  [释放显存] 卸载 SAM 2

Phase 3: 导出
  掩码外区域 → 透明 (alpha=0)
    ↓
  裁剪到掩码边界 + padding
    ↓
  每个元素 → 独立的透明PNG文件
  附加: _preview.png（检测框预览）+ manifest.json（清单）
显存管理：两个模型不会同时驻留显存。Phase 1 完成后释放 Grounding DINO，再加载 SAM 2。峰值显存 ~1.5GB。

7. 自定义提示词指南

核心原则：提示词用英文，用 . 分隔，每个词描述一种元素类型。越具体越好。

游戏UI常用提示词模板

场景推荐提示词
登录奖励day label . reward icon . claim button . gold coin . gem . chest
商城面板price tag . buy button . item icon . currency icon . discount badge . close button
战斗HUDhealth bar . mana bar . skill icon . avatar . level badge . score label
设置面板toggle switch . slider . dropdown . checkbox . close button . save button
背包/背包item slot . item icon . rarity badge . count label . tab button . sort button
通用button . icon . panel . badge . label . bar . frame . decoration . arrow . star . coin . gem . chest . avatar

调参技巧

问题解决方案
检测太少降低 --box_threshold（如 0.15），增加提示词种类
检测太多/误检升高 --box_threshold（如 0.4），提示词写更具体
金币没检测到提示词加 gold coin . coin . circle icon . yellow circle
边界不紧减小 --padding（如 1-2）
边界太紧增大 --padding（如 8-12）
相似元素混淆用更具体的描述：blue button . red button 而非 button . button

8. 常见问题

Q: 报错 "未检测到CUDA GPU"

A: PyTorch 是 CPU 版本。重装 CUDA 版：

pip uninstall torch torchvision
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu128

Q: 报错 CUDA out of memory

A: 显存不足。确保没有其他程序占用GPU，然后：

# 检查显存占用
nvidia-smi

# 如果其他程序占用了显存，先关闭它们
# 特别是浏览器、Cocos Creator等

Q: 模型下载慢 / 下载失败

A: 需要访问 HuggingFace。如果网络受限：

# 方法1: 设置镜像
set HF_ENDPOINT=https://hf-mirror.com

# 方法2: 手动下载模型文件放到缓存目录
# Windows: C:\Users\你的用户名\.cache\huggingface\hub\

Q: 检测到0个元素

A: 提示词与图片内容不匹配。尝试：

# 1. 降低阈值
python ui_slicer_pro.py "图片.png" --box_threshold 0.1 --text_threshold 0.1

# 2. 使用更通用的提示词
python ui_slicer_pro.py "图片.png" -p "object . element . shape . icon . button"

Q: 同类元素被分成不同的label

A: Grounding DINO 的文本匹配会根据提示词分词。如果提示词是 gold coin . blue coin，两类金币会得到不同标签。如果想要统一的标签，用单一提示词 coin。

9. 新旧方案对比

维度 旧方案 (AI坐标) 新方案 (GD+SAM2)
定位方式 AI大模型返回JSON坐标数字 专业检测模型 + 分割模型
输出格式 矩形 bbox (近似坐标) 像素级掩码 + 矩形bbox
圆形/不规则形状 ❌ 按矩形裁切，会有多余背景 ✅ 像素级掩码，精确到边缘
透明背景 ❌ 需要额外的去背景处理 ✅ 掩码外自动透明
金币精度 经常切偏 像素级精确
需要API ❌ 需要调用云端AI API ✅ 完全本地运行
成本 每次调用消耗API token 免费（一次性下载模型）
速度 慢（网络请求 + 多轮重试） 快（本地GPU推理，单次完成）
可定制性 只能通过 prompt 调整 文本提示词 + 阈值 + padding 全可调
网络依赖 ❌ 每次都需要 ✅ 仅首次下载模型
最小显存 不需要GPU ~2GB（顺序加载模式）

#!/usr/bin/env python3
"""
UI切图专业工具 v2.0 - Grounding DINO + SAM 2
==========================================
基于语义分割的像素级UI元素自动切片工具

核心优势（vs 旧版AI坐标方案）：
  - 文本引导检测：用自然语言描述要切的元素类型
  - 像素级掩码：圆形金币、不规则装饰都能精确裁切
  - 透明PNG导出：掩码外区域自动变透明
  - 顺序加载：Grounding DINO 和 SAM 2 不会同时驻留显存，2GB即可跑

用法：
  python ui_slicer_pro.py "图片路径" [--prompt "button . icon . coin"] [--output 输出目录]
"""

def phase1_detect(image_path, text_prompt, box_threshold=0.3, text_threshold=0.25):
    """
    Phase 1: Grounding DINO 文本引导检测
    输入图片 + 文本提示（如 "button . icon . coin"），返回精确边界框
    """
    print("[Phase 1] Grounding DINO 文本引导检测")
    print("-" * 40)

    from transformers import AutoProcessor, AutoModelForZeroShotObjectDetection

    device = "cuda"
    model_id = "IDEA-Research/grounding-dino-tiny"

    print(f"  加载模型: {model_id} (首次运行需下载，约600MB)")
    t0 = time.time()
    processor = AutoProcessor.from_pretrained(model_id)
    model = AutoModelForZeroShotObjectDetection.from_pretrained(model_id).to(device)
    print(f"  模型就绪: {time.time() - t0:.1f}s")

    image = Image.open(image_path).convert("RGB")
    w, h = image.size
    print(f"  图片: {w}x{h}")
    print(f"  检测提示: {text_prompt}")

    inputs = processor(images=image, text=text_prompt, return_tensors="pt").to(device)

    t0 = time.time()
    with torch.no_grad():
        outputs = model(**inputs)
    print(f"  推理完成: {time.time() - t0:.1f}s")

    results = processor.post_process_grounded_object_detection(
        outputs,
        inputs.input_ids,
        threshold=box_threshold,
        text_threshold=text_threshold,
        target_sizes=[(h, w)],
    )

    boxes = results[0]["boxes"].cpu().numpy()  # (N, 4) xyxy
    labels = list(results[0]["labels"])
    scores = results[0]["scores"].cpu().numpy()

    print(f"  检测到 {len(labels)} 个元素:")
    for i, (label, score) in enumerate(zip(labels, scores)):
        x1, y1, x2, y2 = boxes[i]
        print(f"    [{i + 1}] {label}  conf={score:.2f}  bbox=[{int(x1)},{int(y1)},{int(x2)},{int(y2)}]")

    # 释放显存
    del model, processor, inputs, outputs
    torch.cuda.empty_cache()
    alloc = torch.cuda.memory_allocated(0) / 1024 ** 3
    print(f"  [显存释放] 当前占用: {alloc:.2f} GB")
    print()

    return boxes, labels, scores


def phase2_segment(image_path, boxes):
    """
    Phase 2: SAM 2 像素级分割
    接收边界框，为每个框生成像素级二值掩码
    """
    print("[Phase 2] SAM 2 像素级分割")
    print("-" * 40)

    from sam2.sam2_image_predictor import SAM2ImagePredictor

    device = "cuda"

    print(f"  加载模型: sam2.1-hiera-tiny (首次运行需下载，约80MB)")
    t0 = time.time()
    predictor = SAM2ImagePredictor.from_pretrained(
        "facebook/sam2.1-hiera-tiny",
        device=device,
    )
    print(f"  模型就绪: {time.time() - t0:.1f}s")

    image = np.array(Image.open(image_path).convert("RGB"))
    predictor.set_image(image)

    all_masks = []
    all_scores = []

    t0 = time.time()
    for i, box in enumerate(boxes):
        masks, scores, _ = predictor.predict(
            box=box[None, :],  # (1, 4) xyxy
            multimask_output=False,
        )
        all_masks.append(masks.squeeze())
        all_scores.append(float(scores.squeeze()))
    elapsed = time.time() - t0

    print(f"  分割完成: {len(all_masks)} 个掩码 ({elapsed:.1f}s)")
    for i, score in enumerate(all_scores):
        print(f"    [{i + 1}] IoU={score:.3f}")

    # 释放显存
    del predictor
    torch.cuda.empty_cache()
    alloc = torch.cuda.memory_allocated(0) / 1024 ** 3
    print(f"  [显存释放] 当前占用: {alloc:.2f} GB")
    print()

    return np.array(all_masks), np.array(all_scores)


def phase3_export(image_path, masks, labels, boxes, scores, output_dir, padding=4):
    """
    Phase 3: 导出透明PNG切片
    将每个掩码区域裁切为独立PNG文件
    """
    print("[Phase 3] 导出切片")
    print("-" * 40)

    img = np.array(Image.open(image_path).convert("RGBA"))
    os.makedirs(output_dir, exist_ok=True)

    name_counter = {}
    slices = []

    for i, (mask, label, box, score) in enumerate(zip(masks, labels, boxes, scores)):
        # 掩码外区域变透明
        element = img.copy()
        element[~mask] = 0

        # 找掩码实际占据的区域（裁掉空白）
        rows = np.any(mask, axis=1)
        cols = np.any(mask, axis=0)

        if not rows.any() or not cols.any():
            print(f"  跳过空掩码: {label}_{i}")
            continue

        y_idx = np.where(rows)[0]
        x_idx = np.where(cols)[0]
        y1, y2 = y_idx[0], y_idx[-1] + 1
        x1, x2 = x_idx[0], x_idx[-1] + 1

        # 加padding
        x1 = max(0, x1 - padding)
        y1 = max(0, y1 - padding)
        x2 = min(img.shape[1], x2 + padding)
        y2 = min(img.shape[0], y2 + padding)

        cropped = element[y1:y2, x1:x2]

        # 跳过过小区域
        if cropped.shape[0] < 5 or cropped.shape[1] < 5:
            continue

        # 生成唯一文件名
        safe_name = re.sub(r'[<>:"/\\|?*]', '_', label)
        if safe_name in name_counter:
            name_counter[safe_name] += 1
            fname = f"{safe_name}_{name_counter[safe_name]}"
        else:
            name_counter[safe_name] = 0
            fname = safe_name

        out_path = os.path.join(output_dir, f"{fname}.png")
        Image.fromarray(cropped).save(out_path)

        slices.append({
            "file": f"{fname}.png",
            "label": label,
            "bbox": {
                "x": int(x1), "y": int(y1),
                "width": int(x2 - x1), "height": int(y2 - y1),
            },
            "mask_iou": round(float(score), 3),
            "size": f"{cropped.shape[1]}x{cropped.shape[0]}",
        })

        print(f"  [{i + 1}/{len(masks)}] {fname}.png "
              f"({cropped.shape[1]}x{cropped.shape[0]}) IoU={score:.2f}")

    return slices


def generate_preview(image_path, slices, masks, output_dir):
    """生成检测框+掩码叠加预览图"""
    img = cv2.imread(image_path)

    # 按label分配颜色
    color_palette = [
        (0, 200, 255), (0, 255, 100), (255, 100, 0), (0, 255, 255),
        (255, 0, 255), (200, 200, 0), (0, 100, 255), (255, 200, 0),
        (128, 0, 255), (0, 128, 255), (255, 128, 0), (128, 255, 0),
    ]
    color_map = {}
    for s in slices:
        if s["label"] not in color_map:
            color_map[s["label"]] = color_palette[len(color_map) % len(color_palette)]

    # 绘制半透明掩码叠加
    overlay = img.copy()
    for idx, s in enumerate(slices):
        mask = masks[idx]
        color = color_map[s["label"]]
        overlay[mask] = overlay[mask] * 0.5 + np.array(color) * 0.5
    img = cv2.addWeighted(overlay, 0.6, img, 0.4, 0)

    # 绘制边界框和标签
    for s in slices:
        bx, by = s["bbox"]["x"], s["bbox"]["y"]
        bw, bh = s["bbox"]["width"], s["bbox"]["height"]
        color = color_map[s["label"]]
        cv2.rectangle(img, (bx, by), (bx + bw, by + bh), color, 2)
        text = f"{s['label']} {s['size']}"
        (tw, th), _ = cv2.getTextSize(text, cv2.FONT_HERSHEY_SIMPLEX, 0.35, 1)
        cv2.rectangle(img, (bx, by - th - 6), (bx + tw + 4, by), color, -1)
        cv2.putText(img, text, (bx + 2, by - 3),
                    cv2.FONT_HERSHEY_SIMPLEX, 0.35, (255, 255, 255), 1)

    preview_path = os.path.join(output_dir, "_preview.png")
    cv2.imwrite(preview_path, img)
    print(f"  预览图: {preview_path}")


def generate_manifest(image_path, slices, output_dir, text_prompt):
    """生成JSON清单文件"""
    w, h = Image.open(image_path).size

    categories = {}
    for s in slices:
        categories[s["label"]] = categories.get(s["label"], 0) + 1

    manifest = {
        "source": os.path.basename(image_path),
        "source_size": {"width": w, "height": h},
        "method": "Grounding DINO + SAM 2",
        "text_prompt": text_prompt,
        "created_at": datetime.now().isoformat(),
        "total_slices": len(slices),
        "categories": categories,
        "slices": slices,
    }

    manifest_path = os.path.join(output_dir, "manifest.json")
    with open(manifest_path, "w", encoding="utf-8") as f:
        json.dump(manifest, f, ensure_ascii=False, indent=2)
    print(f"  清单: {manifest_path}")


def main():
    parser = argparse.ArgumentParser(
        description="UI切图专业工具 v2.0 - Grounding DINO + SAM 2 语义分割"
    )
    parser.add_argument("image", help="输入图片路径")
    parser.add_argument(
        "--prompt", "-p", default=DEFAULT_PROMPT,
        help="检测提示词，用 . 分隔不同元素类型",
    )
    parser.add_argument("--output", "-o", default=None, help="输出目录（默认=图片名_sliced）")
    parser.add_argument("--padding", "-pd", type=int, default=4, help="外扩边距 px（默认4）")
    parser.add_argument("--box_threshold", "-bt", type=float, default=0.3,
                        help="检测框置信度阈值（默认0.3，降低可检测更多元素）")
    parser.add_argument("--text_threshold", "-tt", type=float, default=0.25,
                        help="文本匹配阈值（默认0.25）")

    args = parser.parse_args()

    if not os.path.exists(args.image):
        print(f"错误: 图片不存在 - {args.image}")
        sys.exit(1)

    # 自动输出目录
    if args.output is None:
        base = Path(args.image).stem
        parent = Path(args.image).parent
        args.output = str(parent / f"{base}_sliced")

    check_env()

    print(f"  输入: {args.image}")
    print(f"  输出: {args.output}")
    print()

    t_total = time.time()

    # Phase 1: 文本引导检测
    boxes, labels, scores = phase1_detect(
        args.image, args.prompt,
        box_threshold=args.box_threshold,
        text_threshold=args.text_threshold,
    )

    if len(boxes) == 0:
        print("未检测到任何元素，建议：")
        print("  1. 降低 --box_threshold（如 0.15）")
        print("  2. 更换 --prompt（更具体的描述，如 'gold coin . blue button'）")
        sys.exit(0)

    # Phase 2: 像素级分割
    masks, mask_scores = phase2_segment(args.image, boxes)

    # Phase 3: 导出
    slices = phase3_export(
        args.image, masks, labels, boxes, mask_scores,
        args.output, args.padding,
    )

    # 生成预览和清单
    print("[输出]")
    generate_preview(args.image, slices, masks, args.output)
    generate_manifest(args.image, slices, args.output, args.prompt)

    # 汇总
    elapsed = time.time() - t_total
    cats = {}
    for s in slices:
        cats[s["label"]] = cats.get(s["label"], 0) + 1

    print()
    print("=" * 60)
    print(f"  切图完成!")
    print(f"  耗时:   {elapsed:.1f}s")
    print(f"  输出:   {args.output}")
    print(f"  切片:   {len(slices)} 个")
    print(f"  分类:   {cats}")
    print("=" * 60)


if __name__ == "__main__":
    main()
